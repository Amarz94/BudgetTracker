## Description

To create an a financial tracker that takes withdrawals and deposits and updates the graph showing the changes in the account.

## Deployment

### Clone Repository 

```bash
git clone git@github.com:Amarz94/BudgetTracker.git

```

### Required Programs

- Node.JS
- MySQL
- NPM

### Required Dependencies

- compression
- express
- mongoose
- morgan

### Launching program

```bash
node server.js

```
